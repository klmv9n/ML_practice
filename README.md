# Описание проекта

Проект направлен на разработку системы для исключения антропогенного воздействия на экосистемы охраняемых природных территорий.

Система при должной степени доработки может быть использована в качестве инструмента контроля за соблюдением режима заповедности, в частности для:

- снижения фактора беспокойства для животных со стороны человека;
- раннего обнаружения незаконной деятельности (присутствие человека в строго заповедных зонах, проезд техники, браконьерство) и т.п.

## Возможный подход к применению готового проекта:

1. Развёртывание фотоловушек (на экотропах, границах и т.п.) в зонах строгой заповедности с минимальным вмешательством в среду.
2. Сбор и, возможно, первичная фильтрация данных.
3. Обработка изображений при помощи используемой модели (детектирование).
4. Принятие мер по минимаизации антропогенного взаимодействия (изменения границ зон, маршрутов экотроп и т.п.)
   или его полное пресечение (задержание нарушителей при начичии оснований для этого).

## Основные цели

1. Найти подходящую предобученную модель, способную детектировать людей и животных на снимках с фотоловушек.

   Было решено воспользоваться фреймворком [PyTorch-Wildlife](https://github.com/microsoft/CameraTraps)
   с открытым исходным кодом. Он специализируется на исследованиях в области компьютерного зрения,
   ориентированных на сохранение биоразнообразия и мониторинг дикой природы.

   Из числа моделей экосистемы была выбрана MegaDetectorv6. Она как раз предназначена для детектирования
   животных, людей и транспорта на фотографиях с камер-ловушек.

2. Подготовить тестовый датасет, состоящий из небольшого числа фотографий, и провалидировать на нём модель. 

   Для проверки работоспособности модели были выбраны фото с животными и людьми.
